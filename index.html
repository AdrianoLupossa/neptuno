Quick introduction

O CSS tem crescido muito ao longo dos anos e esta biblioteca possuí alguns componentes que antes eram possiveis apenas com javascript e hoje são possíveis só CSS.

Neptuno é uma biblioteca com um conjunto de componentes criados com CSS puro, sem javascript.

Neptuno é super customizável, eu preciso apenas ter conhecimento de HTML & CSS para poder personalizar qualquer componente de acordo com as minhas preferências.

São componentes extremamente performáticos, afinal é apenas HTML e CSS.

Os componentes são extremamente compatíveis com a maioria dos navegadores.

Todos os componentes foram construídos de forma a garantir uma boa experiência aos usuários.

Você pode apenas pegar os componentes que precisas e junta-los as tuas próprias folhas de estilo, é incrível.
