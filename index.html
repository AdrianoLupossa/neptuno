Quick introduction

CSS has grown a lot over the years and this library has some components that were previously only possible with javascript and today are possible with CSS.

Neptune is a library with a set of components created with pure CSS, without javascript.

is super customizable, I just need to know HTML & CSS to be able to customize any component according to my preferences.

They are extremely performative components, after all it's just HTML and CSS.

The components are extremely compatible with most browsers.

All components were built to ensure a good user experience.

You can just pick up the components you need and add them to your own style sheets, it's amazing.
